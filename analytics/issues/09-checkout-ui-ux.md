# Issue 09: UI/UX de Checkout e Estados de Carregamento

## Descrição
Melhorar a percepção de performance e confiança do usuário final durante o fluxo de pagamento.

## Tarefas
- [ ] Implementar *Skeleton Loaders* na transição para o Checkout.
- [ ] Adicionar modal de feedback rico (Success/Error) com ícones animados da `motion`.
- [ ] Refinar redirecionamento após pagamento Pix (exibir QR Code com timer de expiração).
- [ ] Adicionar botão de "Ajuda via WhatsApp" visível durante o checkout.

## Critérios de Aceite
- O usuário nunca fica travado em tela branca durante a geração de assinaturas.
- Feedback visual claro sobre o sucesso ou falha da transação.
